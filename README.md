# Robotics Arena - BotLeague

Frontend implementation of BotLeague landing page.

**Stack:** React 18 + TypeScript + Tailwind CSS + Vite

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Sections
- Navbar (fixed, mobile responsive)
- Hero (full-screen with CTA)
- Competitions & Events (tabbed: Live / Upcoming / Past)
- User Journey (4 steps)
- What is BotLeague? (4 features)
- Categories (4 cards)
- Competition Disciplines (6 sports)
- League Advantage + Leaderboard
- Join the Ecosystem (3 signup forms)
- Sponsors
- Footer

> Frontend only. No backend. No API calls. All data is static.# BotLeague
